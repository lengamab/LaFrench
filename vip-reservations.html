Informations Importantes

Tout ce qu'il faut savoir avant de réserver

Qu'est-ce qui est inclus dans le prix ?

Le montant payé est un "minimum de consommation". Cela signifie que l'intégralité de la somme est transformée en crédit pour commander des bouteilles ou des boissons à la carte.

Quelle est l'heure limite d'arrivée ?

Pour garantir votre table, une arrivée avant 01h30 est fortement recommandée. Après cette heure, le club se réserve le droit de réattribuer la table en cas de forte affluence.

Quel est le dress code ?

Pour les zones VIP, une tenue élégante est obligatoire. Chemise recommandée pour les hommes, chaussures de ville (pas de baskets de sport). Le club se réserve le droit d'entrée.
